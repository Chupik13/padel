## [1.2.0] — 2026-03-04

### Добавлено

- **Статистика побед с партнёрами** — в профиле внутри каждого блока (текущий сезон, общая статистика, прошлые сезоны) появился collapse-список партнёров с процентом побед, количеством совместных игр. Сортировка по winRate.
- **Личная встреча (Head-to-Head)** — при просмотре чужого профиля отображается карточка сравнения: матчи как соперники (бар побед), матчи как партнёры (winRate, игры вместе), средние очки. Новый эндпоинт `GET /api/profile/head-to-head/{login}`.
- **Быстрый реванш** — после завершения live-турнира хост видит кнопку «Реванш», которая создаёт новый турнир с теми же игроками и форматом.
- **График очков сезона** — на странице сезонов клик по строке игрока показывает bar chart с результатами по турнирам. Зелёные бары — учтённые в рейтинге, серые — не учтённые. Данные приходят с бэкенда (`TournamentScores` в `PlayerSeasonScoreResult`).
- **Обратный отсчёт сезона** — на карточке текущего сезона badge с количеством дней до конца (с плюрализацией RU/EN).

## [1.0.0] — 2026-03-03

Первый релиз приложения для управления турнирами по падел-теннису.
C#/.NET 10 бэкенд + React/TypeScript фронтенд.

### Добавлено

- **Сжатие аватарок** — автоматическое сжатие до 256x256 и сохранение в WebP (SixLabors.ImageSharp). Миграция существующих PNG/JPG при старте сервера.
- **Сбалансированные форматы турниров** — замена фиксированных опций (5/10) на Баланс/Малый/Средний с расчётом по формуле `C(n,2) * k / 2`:
  - 4 игрока: Баланс(3), Малый(6), Средний(9)
  - 5 игроков: Баланс(5), Средний(10)
  - 6 игроков: Баланс(15), Малый(5), Средний(10)
- **Досрочное завершение турнира** — кнопка "Завершить досрочно" для товарищеских игр. Неотыгранные матчи получают счёт 8:8. Модальное подтверждение, синхронизация через SignalR. Для сезонных игр недоступно.
- **Общий лидерборд** — вкладки "История турниров" / "Общий рейтинг" на странице турниров. Сводная таблица: игры, очки, средние очки (общие и сезонные). Ленивая загрузка. Endpoint `GET /api/players/leaderboard`.

### Исправлено

- Битые турниры (без матчей) скрыты из всех списков и автоматически отменяются
- Защита от краша `MatchView` при некорректном `currentMatchIndex`
- Корректная установка `inSeason` при загрузке активного турнира с сервера
