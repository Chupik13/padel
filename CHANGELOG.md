# Changelog

## v1.0.0 (2026-03-03)

### Сжатие аватарок
- Аватарки автоматически сжимаются до 256x256 и сохраняются в формате WebP
- При старте сервера существующие PNG/JPG аватарки мигрируются в WebP
- Добавлен SixLabors.ImageSharp для обработки изображений

### Сбалансированные форматы турниров
- Заменены фиксированные опции (5/10) на Баланс/Малый/Средний
- 4 игрока: Баланс(3), Малый(6), Средний(9) — все сбалансированные
- 5 игроков: Баланс(5), Средний(10) — Малый скрыт (равен Балансу)
- 6 игроков: Баланс(15), Малый(5), Средний(10)

### Досрочное завершение турнира
- Кнопка "Завершить досрочно" (только товарищеские игры)
- Все неотыгранные матчи получают счёт 8:8
- Модальное подтверждение, синхронизация через SignalR
- Для сезонных игр досрочное завершение недоступно

### Общий лидерборд
- Вкладки на странице турниров: "История турниров" и "Общий рейтинг"
- Сводная таблица со статистикой по всем и сезонным играм
- Ленивая загрузка при переключении на вкладку рейтинга
- Backend endpoint GET /api/players/leaderboard

### Исправления
- Битые турниры (без матчей) скрыты из всех списков
- Защита от краша MatchView при некорректном currentMatchIndex
