## [2.0.0] — 2026-03-15

### Добавлено

- **Поддержка 7 игроков** — теперь можно играть вчетвером, впятером, вшестером и всемером.
  - Сезонная игра: 21 матч (balanced, k=2).
  - Товарищеская игра: 7 (fixed), 14 (fixed), 21 (balanced) матчей на выбор.
  - Каждый матч: 4 играют, 3 отдыхают.

### Улучшено

- **Навигация во время турнира** — кнопки «Назад» и «Далее» скрыты на вкладке «Турнир», отображаются только на вкладке «Матч».

## [1.3.1] — 2026-03-13

### Добавлено

- **Мультиклубы** — игрок может состоять в нескольких клубах одновременно. В настройках можно выбрать основной клуб. Статистика, сезоны и турниры привязаны к основному клубу.
- **Выбор клуба при создании турнира** — на странице «Играть» можно выбрать клуб, в котором будет создан турнир (если клубов несколько).
- **Страница клубов переработана** — секции «Мои клубы» и «Все клубы», вступление без выхода из текущего, меню с действиями (сделать основным / покинуть).

### Улучшено

- **Упрощён процесс создания турнира** — убран отдельный шаг выбора количества игроков, количество определяется автоматически из выбранных (4–6). Новый порядок: клуб → тип игры → игроки → количество матчей → старт.
- **Настройки профиля** — добавлен выбор основного клуба (при наличии нескольких).

### Исправлено

- **Статистика клуба** — рейтинг теперь корректно фильтруется по выбранному клубу, а не показывает одинаковые данные во всех клубах.
- **Генерация расписания на 4 игрока** — исправлена ошибка «нет матчей» при создании турнира на 4 игрока с повторами.
- **Обновление счётчика участников** — количество участников клуба обновляется сразу после вступления без перезагрузки страницы.
- **Создание клуба не меняет основной** — создание нового клуба больше не перезаписывает основной клуб, если он уже установлен.
- **Запрет турниров без клуба** — нельзя создать турнир, если игрок не состоит ни в одном клубе.

## [1.3.0] — 2026-03-13

### Добавлено

- **Форма обратной связи** — вместо статичной страницы контактов теперь полноценная форма с выбором темы (предложение, баг, другое), полем сообщения и опциональным email для ответа. Письмо отправляется на почту разработчика.
- **Таймеры матчей и турниров** — во время игры отображается время текущего матча и общее время турнира. В истории турниров рядом с каждым матчем показана его длительность.
- **Длительность турнира в истории** — в карточке турнира рядом с количеством игроков отображается общее время (для завершённых — статичное, для текущих — в реальном времени).
- **Фильтры турниров** — на странице истории появились фильтры по типу (сезонная/товарищеская), досрочно завершённые и отменённые.
- **Удаление турниров** — администратор может безвозвратно удалить турнир из истории.
- **Наблюдение за играми клуба** — на странице «Играть» видны текущие турниры других участников клуба с возможностью наблюдения.
- **Настройки профиля** — страница для смены имени, доступна из профиля.
- **Автоочистка зависших турниров** — турниры, где текущий матч не обновлялся более 40 минут, автоматически завершаются досрочно или отменяются.
- **Сезоны** — карточки будущих сезонов, обратный отсчёт до конца текущего сезона, скрытие топ-3 из таблицы после супер-игры.

### Улучшено

- **Распределение отдыха** — в турнирах на 5-6 игроков отдых распределяется равномернее, игроки не играют слишком много матчей подряд.
- **Порядок в команде** — игроки чередуются в позиции «первый/второй» в команде, а не всегда в одном порядке.
- **Хост отмечен точкой** — вместо эмодзи короны рядом с именем хоста теперь аккуратная золотая точка.
- **Минимум 60% матчей для досрочного завершения** — нельзя завершить турнир досрочно, если сыграно менее 60% матчей.
- **Аватары в результатах** — в таблице результатов турнира теперь отображаются аватары игроков.
- **Незавершённые турниры исключены из статистики** — в рейтинг и статистику попадают только полностью завершённые турниры.
- **Ссылка на восстановление пароля действительна 1 час** (было 30 минут).

### Исправлено

- **Удалены debug-логи** — убраны отладочные Console.WriteLine из production-кода.

## [1.2.0] — 2026-03-04

### Добавлено

- **Клубы** — система клубов для организации групп игроков. Можно создать свой клуб или вступить в существующий. Все турниры, сезоны и рейтинги привязаны к клубу. При первом входе предлагается выбрать или создать клуб.
- **Локализация RU/EN** — полный перевод интерфейса на русский и английский. Переключение языка в шапке и на страницах логина/регистрации. Язык определяется автоматически по браузеру, сохраняется в localStorage.
- **Статистика побед с партнёрами** — в профиле внутри каждого блока (текущий сезон, общая статистика, прошлые сезоны) появился раскрывающийся список партнёров с процентом побед и количеством совместных игр.
- **Личная встреча (Head-to-Head)** — при просмотре чужого профиля отображается карточка сравнения: матчи как соперники, матчи как партнёры, средние очки.
- **Быстрый реванш** — после завершения live-турнира хост видит кнопку «Реванш», которая создаёт новый турнир с теми же игроками и форматом.
- **График очков сезона** — на странице сезонов клик по строке игрока показывает график с результатами по турнирам. Зелёные столбцы — учтённые в рейтинге, серые — не учтённые.
- **Обратный отсчёт сезона** — на карточке текущего сезона отображается количество дней до конца.
- **Автопереход на «Что нового»** — при обновлении версии приложение автоматически открывает страницу с последними изменениями.
- **Подсказки** — иконки «i» на ключевых страницах с пояснениями по механикам (сезоны, турниры, профиль, клубы).

### Изменено

- **Страница «Что нового»** — changelog загружается с GitHub (CHANGELOG.md / CHANGELOG.en.md) вместо захардкоженного списка. Версии группируются по major.minor, новая версия подсвечивается при переходе.
- **Сезоны привязаны к клубу** — лидерборд и статистика сезона фильтруются по клубу текущего пользователя.
- **Турниры привязаны к клубу** — история турниров и лидерборд показывают только данные своего клуба.
- **Новая иконка приложения** — фавикон заменён на SVG-логотип.

## [1.0.0] — 2026-03-03

Первый релиз приложения для управления турнирами по падел-теннису.
C#/.NET 10 бэкенд + React/TypeScript фронтенд.

### Добавлено

- **Сжатие аватарок** — автоматическое сжатие до 256x256 и сохранение в WebP (SixLabors.ImageSharp). Миграция существующих PNG/JPG при старте сервера.
- **Сбалансированные форматы турниров** — замена фиксированных опций (5/10) на Баланс/Малый/Средний с расчётом по формуле `C(n,2) * k / 2`:
  - 4 игрока: Баланс(3), Малый(6), Средний(9)
  - 5 игроков: Баланс(5), Средний(10)
  - 6 игроков: Баланс(15), Малый(5), Средний(10)
- **Досрочное завершение турнира** — кнопка "Завершить досрочно" для товарищеских игр. Неотыгранные матчи получают счёт 8:8. Модальное подтверждение, синхронизация через SignalR. Для сезонных игр недоступно.
- **Общий лидерборд** — вкладки "История турниров" / "Общий рейтинг" на странице турниров. Сводная таблица: игры, очки, средние очки (общие и сезонные). Ленивая загрузка. Endpoint `GET /api/players/leaderboard`.

### Исправлено

- Битые турниры (без матчей) скрыты из всех списков и автоматически отменяются
- Защита от краша `MatchView` при некорректном `currentMatchIndex`
- Корректная установка `inSeason` при загрузке активного турнира с сервера
