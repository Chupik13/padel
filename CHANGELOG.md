## [1.2.1] — 2026-03-04

### Добавлено

- **Восстановление пароля через email** — на странице логина появилась ссылка «Забыли пароль?». Ввод логина → письмо со ссылкой для сброса → страница ввода нового пароля. Отправка через Gmail SMTP.
- **Email при регистрации** — поле email теперь обязательно при создании аккаунта.
- **Модалка «Укажите email»** — существующие пользователи без email увидят модальное окно с просьбой указать почту (для восстановления пароля).

### Исправлено

- **Тултипы не выходят за экран** — подсказки (i) теперь всегда позиционируются в пределах видимой области, не обрезаются контейнерами с overflow.

## [1.2.0] — 2026-03-04

### Добавлено

- **Клубы** — система клубов для организации групп игроков. Можно создать свой клуб или вступить в существующий. Все турниры, сезоны и рейтинги привязаны к клубу. При первом входе предлагается выбрать или создать клуб.
- **Локализация RU/EN** — полный перевод интерфейса на русский и английский. Переключение языка в шапке и на страницах логина/регистрации. Язык определяется автоматически по браузеру, сохраняется в localStorage.
- **Статистика побед с партнёрами** — в профиле внутри каждого блока (текущий сезон, общая статистика, прошлые сезоны) появился раскрывающийся список партнёров с процентом побед и количеством совместных игр.
- **Личная встреча (Head-to-Head)** — при просмотре чужого профиля отображается карточка сравнения: матчи как соперники, матчи как партнёры, средние очки.
- **Быстрый реванш** — после завершения live-турнира хост видит кнопку «Реванш», которая создаёт новый турнир с теми же игроками и форматом.
- **График очков сезона** — на странице сезонов клик по строке игрока показывает график с результатами по турнирам. Зелёные столбцы — учтённые в рейтинге, серые — не учтённые.
- **Обратный отсчёт сезона** — на карточке текущего сезона отображается количество дней до конца.
- **Автопереход на «Что нового»** — при обновлении версии приложение автоматически открывает страницу с последними изменениями.
- **Подсказки** — иконки «i» на ключевых страницах с пояснениями по механикам (сезоны, турниры, профиль, клубы).

### Изменено

- **Страница «Что нового»** — changelog загружается с GitHub (CHANGELOG.md / CHANGELOG.en.md) вместо захардкоженного списка. Версии группируются по major.minor, новая версия подсвечивается при переходе.
- **Сезоны привязаны к клубу** — лидерборд и статистика сезона фильтруются по клубу текущего пользователя.
- **Турниры привязаны к клубу** — история турниров и лидерборд показывают только данные своего клуба.
- **Новая иконка приложения** — фавикон заменён на SVG-логотип.

## [1.0.0] — 2026-03-03

Первый релиз приложения для управления турнирами по падел-теннису.
C#/.NET 10 бэкенд + React/TypeScript фронтенд.

### Добавлено

- **Сжатие аватарок** — автоматическое сжатие до 256x256 и сохранение в WebP (SixLabors.ImageSharp). Миграция существующих PNG/JPG при старте сервера.
- **Сбалансированные форматы турниров** — замена фиксированных опций (5/10) на Баланс/Малый/Средний с расчётом по формуле `C(n,2) * k / 2`:
  - 4 игрока: Баланс(3), Малый(6), Средний(9)
  - 5 игроков: Баланс(5), Средний(10)
  - 6 игроков: Баланс(15), Малый(5), Средний(10)
- **Досрочное завершение турнира** — кнопка "Завершить досрочно" для товарищеских игр. Неотыгранные матчи получают счёт 8:8. Модальное подтверждение, синхронизация через SignalR. Для сезонных игр недоступно.
- **Общий лидерборд** — вкладки "История турниров" / "Общий рейтинг" на странице турниров. Сводная таблица: игры, очки, средние очки (общие и сезонные). Ленивая загрузка. Endpoint `GET /api/players/leaderboard`.

### Исправлено

- Битые турниры (без матчей) скрыты из всех списков и автоматически отменяются
- Защита от краша `MatchView` при некорректном `currentMatchIndex`
- Корректная установка `inSeason` при загрузке активного турнира с сервера
